# Niji Editor

An extremely weird and dramatic editor, based on (by that i mean copy pasted) from [Tsoding's ded](https://github.com/tsoding/ded).

## Quick Start

## POSIX

```console
$ make
$ ./niji src/main.c
```

## Windows MSVC

```console
> .\setup_dependencies.bat
> .\build_msvc.bat
> .\niji.exe src\main.c
```
